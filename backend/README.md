3.0.2

# How to Run:

Add a local_env.yml file with keys:

- OPENAI_ACCESS_TOKEN: "your openai secret key"
- REQUEST_TIMEOUT: "number"

In your console run:
`bundle install`

Then you should be good to do:
`rails s`

After this the API should be ready to go üëç
